# 📈 AI Sales Prediction & Marketing Analytics Dashboard

A Machine Learning-powered web application that predicts future sales based on advertising investments, customer targeting, and marketing strategies. This project combines data analysis, predictive modeling, and an interactive Flask dashboard to help businesses make data-driven marketing decisions.

---

## 🚀 Project Overview

The **AI Sales Prediction & Marketing Analytics Dashboard** is designed to forecast sales using historical marketing data and machine learning algorithms. Users can enter advertising budgets and campaign details through a modern web interface, and the application predicts expected sales while providing business insights and performance recommendations.

---

## 🎯 Objectives

* Predict future sales using Machine Learning.
* Analyze the impact of advertising investments on sales performance.
* Compare different marketing channels.
* Help businesses optimize marketing strategies.
* Present predictions through an interactive dashboard.
