# 📊 Global Unemployment Analytics Dashboard

## Overview

This project was completed as the **second task** during my **Data Analytics Internship at CodeAlpha**.

The project analyzes global unemployment data from **2014 to 2024** using Python. It includes data cleaning, exploratory data analysis (EDA), interactive visualizations, and a professional multi-page Streamlit dashboard to explore unemployment trends across countries, genders, and age groups.

---

## Objectives

* Clean and preprocess unemployment data.
* Explore unemployment trends over time.
* Analyze the impact of Covid-19 on unemployment.
* Compare unemployment across countries, genders, and age groups.
* Identify important trends and patterns.
* Build an interactive Streamlit dashboard.
* Generate insights that can support economic and social policy decisions.
