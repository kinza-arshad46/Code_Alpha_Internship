# 🌸 Smart Iris Flower Classification

## 📌 Overview

This project was developed as **Task 01** for the **Code Alpha Data Science Internship**. It is an interactive machine learning application that classifies Iris flower species based on user-provided flower measurements through a clean and responsive Streamlit dashboard.

## ✨ Features

* Interactive Streamlit Dashboard
* Iris Species Prediction
* Exploratory Data Analysis (EDA)
* Data Visualization
* Correlation Heatmap
* Feature Distribution Analysis
* Confusion Matrix
* Classification Report
* Model Performance Evaluation
* Responsive and User-Friendly Interface

## 🛠️ Technologies Used

* Python
* Streamlit
* Scikit-learn
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Pickle

## 📂 Project Structure

* `app.py` – Main Streamlit application
* `iris_model.pkl` – Trained machine learning model
* `iris.csv` – Dataset
* `requirements.txt` – Project dependencies

## 🎯 Objective

To build a practical machine learning application that combines predictive modeling with interactive data visualization, providing users with an intuitive experience for Iris flower classification.

## 🚀 Outcome

This project demonstrates the complete workflow of a machine learning solution, including data exploration, model deployment, visualization, and an interactive user interface.
